!add

Add a role, channel, or moderator.

!add role <role_name>
!add channel <channel_name>
!add moderator @user

!remove

Remove a role, channel, or moderator.

!remove role <role_name>
!remove channel <channel_name>
!remove moderator @user

!clear

Clear a specific number of messages or all messages.

!clear <number>
!clear all

!unban

Unban a user by their ID or unban all users.

!unban <user_id>
!unban all

Owner and Moderator Commands

Only server owners and designated moderators can use the commands listed above. The bot ensures this by checking user roles and permissions.
