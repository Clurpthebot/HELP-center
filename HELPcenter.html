Introduction

Welcome to the Clurp the bot HELP Center. Here you can find information about how to use the various commands and features of the bot.

Getting Started

To get started with Clurp, invite the bot to your server and use the !help command to see a list of available commands.

Commands

!add

Add a role, channel, or moderator.

!add role
!add channel
!add moderator @user

!remove

Remove a role, channel, or moderator.

!remove role
!remove channel
!remove moderator @user

!clear

Clear a specific number of messages or all messages.

!clear
!clear all

!unban

Unban a user by their ID or unban all users.

!unban
!unban all

Owner and Moderator Commands

Only server owners and designated moderators can use the commands listed above. The bot ensures this by checking user roles and permissions.
